# 视频提示词生成规则

> 适用范围：drama-generate 阶段（将剧本转化为 Seedance/Doubao 视频提示词）
> 核心原则：1 Scene = 1 Shot，不拆分

---

## 零、两阶段分工

| 阶段 | 产出方 | 职责 |
|------|--------|------|
| drama-write | 剧本→JSON | 输出基础 positive（纯动作/场景/运镜/光线），不含参考图声明、不含外貌形容词 |
| video-generate | CLI自动补全 | 从 assets.json 查 TOS URL，在 positive 头部注入参考图声明 |

**drama-write 编写 positive 规则**：
- 只写动作、场景、运镜、光线，**禁止外貌形容词**（枯瘦、花白、骨瘦如柴等）
- `character_references` 中的 name 必须与 assets.json 中角色名精确匹配
- `prop_references` 中的 name 必须与 assets.json 中道具名精确匹配
- 每个镜头加 `spatial_position` 字段标注角色在画面中的大致位置（如"画面中景病床上"）

**video-generate 自动补全格式**（遵循官方 @图片N 标签式规范）：
```
{角色A名} @图片2。位于{spatial_position}。背景{场景名} @图片1。道具{道具名} @图片3在{位置}。

{base_positive}
```

编号顺序固定：场景(@图片1) → 角色(@图片2...) → 道具(@图片N+1...)

规则：
- `@图片N` 标签贴附在元素名**后面**（标签式，非前缀式），空格可有可无
- 编号顺序固定：场景 → 角色 → 道具
- 禁止孤立堆叠多个 @图片N，必须绑定到具体元素名
- 每个元素的 @图片N 放在该元素**首次出现**处
- **道具参考图适用条件**：道具在镜头中作为特写主体或被角色手持交互时才加 @图片声明。仅作为背景陈设的道具不加

## 第一部分：八维视频提示词框架

### 1. 主体描述

- 角色位置（左三分点/右三分点/居中/中景病床上等空间坐标），凡出现角色用正式姓名
- **外观、体型、着装以参考图为准，禁止在提示词中重复描述**（避免与参考图冲突导致AI二次加工跑偏）
- 只写动作和表情，不写长相
- 面部表情与神态（必须描述至少一种具体面部活动）：
  - 眼神（凝视/躲闪/空洞/锐利）
  - 嘴角（紧抿/微扬/颤抖）
  - 眉头（紧锁/舒展/微蹙）
  - 呼吸节奏（急促/屏息/深呼吸）
- 体态与身体语言：肩膀松垮/紧绷、双手活动、身体倾向
- **动作必有从属主语**：每个身体部位前加属主代词，禁止凭空出现的肢体
  - ✗ "一只手入镜关掉闹钟" ← 幽灵手
  - ✗ "拇指滑过屏幕" ← 谁的拇指？
  - ✓ "她的右手拇指滑过手机屏幕"
- **精确到具体部位**：写 `右手食指` 不写 `手`，写 `左手掌心` 不写 `手`
- 示例："眉头紧锁，嘴唇抿成一条线，右手不自觉地攥紧衣角"（不写外貌）

### 2. 动作/运动

- 从原始 action 提取核心动作，可细化但不能臆造
- **优先低缓连续小动作**：优先选择缓慢、连续的小幅度动作，而非大动作或静止。动作之间补充过渡衔接
  - ✗ "她猛地转头，然后立刻站起" ← 无过渡
  - ✓ "她缓缓转过头，看了一眼门口，双手撑着桌面慢慢站起身" ← 有过渡
- 对白必须附带情感演绎方式，禁止只写对白文本
- 对白使用花括号 `{对白内容}` 格式（Seedance 台词规范）：
  - ✗ "林若雪说：'我知道真相。'"
  - ✓ 林若雪深吸一口气，声音压得很低但每个字都在发抖：{我知道真相。}
- 情感状态通过身体动作外化
- 音效使用尖括号 `<远处传来狗叫声>`，音乐使用圆括号 `（背景中播放快节奏BGM）`

### 3. 场景环境与空间锚定

- 场景环境和空间布局通过参考图表达，减少文字描述（Seedance 推荐做法）
- 只写镜头中变化的部分（光线变化、道具位移），不重复描述参考图已包含的静态环境
- **空间锚定**：必须写一句角色在场景中的位置（"位于画面中景病床上""站在灶台前""蹲在门口右侧"）
- **角色-场景交互**：描述角色与场景关键物体的接触和遮挡关系（"被子盖至胸口""手指抠紧门框""背撞上灶台边缘"）
- **身体结构提醒**：被子/衣物覆盖身体时，加结构提示避免AI抹平轮廓（"被子自然隆起显出腿的轮廓"）
- 场景切换时必须有过波描述

#### 3.1 同场景跨镜头站位一致性

> **核心问题**：Seedance 无法跨视频保持人物位置。如果把一段连续戏拆成多个 4s 视频分别生成，
> 每个视频的 AI 会重新"猜"人物站位，导致拼接后人物跳位、物体漂移、背景不一致。
> **解决**：同空间连续动作尽量用一个长镜头（6-12s）生成，减少跨视频拼接。

**如必须拆镜**，以下规则保护站位连续性：

1. **共享地标键名**：同场景所有镜头的 `spatial_anchors` 必须使用相同的键名集合（如 `bar_counter`, `window_seat`, `front_door`），仅位置描述按镜头变化。这作为 Seedance 的稳定空间参照系。

2. **站位继承声明**：每个非首镜头的 positive 开头必须声明上一镜结束时的站位：
   ```
   [角色A]仍站在吧台后，双手保持撑在台面上的姿势。[本镜头动作开始]
   ```

3. **禁止凭空跳位**：角色不能在不切镜的情况下从画面左侧跳到右侧。如有位置变化，必须有过渡动作（走位/转身/坐下/站起）。

4. **物体锚定**：场景物体（固定物+道具）在每个镜头的 positive 中至少提及一次，作为空间参照点。分两类处理：

   **固定物体**（墙壁、门窗、吧台、柱子、楼梯、大型家具等——不会移动的）：
   - 首镜头必须列出关键固定物的画面位置：`吧台横贯画面中景`、`右侧墙面有落地窗`、`后景楼梯通向二楼`
   - 后续同场景镜头必须沿用这些固定物的位置，不得出现"吧台突然消失"或"落地窗变成砖墙"
   - 固定物的空间关系构成场景的"骨架"，Seedance 以此为参照放置人物和道具

   **道具**（杯子、手机、文件、包、钥匙等——可被角色移动的）：
   - 每个镜头必须明确道具的当前状态：在谁手里 / 在哪个固定物上 / 在地上
   - 道具跨镜位移必须有动作交代：上一镜在桌上 → 本镜角色拿起 → 在角色手中
   - 禁止凭空出现或消失：上一镜桌上的杯子不能下一镜突然不见（除非有动作合理性，如被打碎/被拿走）

5. **站位检查清单**（每个同场景连续镜头对）：
   - [ ] 上一镜结束位置 = 本镜开始位置？
   - [ ] 关键固定物体位置一致？（吧台/门窗/楼梯还在原位？）
   - [ ] 道具状态接续？（杯子在谁手里/桌上？文件翻到哪页？）
   - [ ] 如有站位变化，过渡动作已写入本镜？
   - [ ] spatial_anchors 键名集合一致？

### 4. 镜头语言

- 景别 + 运镜 + 拍摄角度
- 禁止写焦段/光圈等相机参数（Seedance 不识别，浪费 token）

### 5. 光影色调

- 三点布光（主光/辅助光/轮廓光），必须指定光源方向和色温

### 6. 风格情绪

- 从风格配置继承，禁止自行发明抽象情绪词

### 7. 时长与节奏

- 用镜头自然节奏推进叙事（如"缓慢推进建立压抑感"）
- 禁止精确到秒的时间分段（如 "0-5s: ..."）。Seedance 对精确时间支持不稳定

**时长估算（按动作数量）**：

| 动作复杂度 | 建议时长 | 示例 |
|-----------|---------|------|
| 单一微动作（手指颤抖/眼眶泛红） | 2-3s | ECU手部特写 |
| 一次完整动作（转身/站起来/关门） | 3-4s | MS中景 |
| 连续2-3步动作（摸出照片→举到眼前→颤抖） | 5-6s | WIDE全景 |
| 多角色交互（对话+反应+走动） | 5-8s | MCU小全清 |
| 一组人的连续对话推进 | 6-10s | 多人对坐/并排/缓步 |
| 一镜完成多步情绪累积 | 8-12s | 推镜头下情绪从平到紧 |

> ⚠️ **Seedance 平台能力**：4–15s 视频生成。**禁止把所有镜头压成 4s**。
> 4s 是"最小切片"，不是"默认值"。动作越轻、越连续、越同空间 → 时长越长。

**时长×动作量配比**（避免 Seedance 崩坏）：

| 时长 | 最多动作步数 | 最多台词句子 |
|------|-------------|-------------|
| 3–4s | 1 步 | 1 句 |
| 5–6s | 2–3 步 | 1–2 句 |
| 6–8s | 3–4 步 | 2 句 |
| 8–10s | 4–5 步 | 2–3 句 |
| 10–12s | 5–6 步 | 3 句 |

> 超过 8s 必须把动作拆成"连续小动作"（微表情+手部+身体倾向），
> 不能塞大动作（猛地站起、冲出门），否则 Seedance 容易崩。

**禁止**：
- 4s 镜头塞 3 步以上动作（动作链太长，必崩）
- 12s+ 镜头无台词/无动作（容易生成空镜）
- 1s/2s 极短镜头（Seedance 难以控制）

### 7.1 镜头合并决策（避免过切）

> 拿到一串候选镜头时，先合并再决定切不切。

**判定顺序**（满足任一就必须保留切分）：

1. **轴线变了吗？** 180° 切换另一侧 → 保留切
2. **空间/机位变了吗？** 换到另一位置/角度 → 保留切
3. **时间跳了吗？** 切完是新时间点 → 保留切
4. **新信息揭示了吗？** 新人物/新道具/新画面 → 保留切
5. **情绪升档了吗？** 从平→紧（更近景别+更快节奏）→ 保留切

5 个都是"否" → **合并到上一个镜头**，让一个 take 跑完。

**反例：第 60 集场景 4 实际分镜**

```
S4_F01 三人晒太阳（MCU，3-4s）
S4_F02 女儿说："我最喜欢的人是爸爸妈妈"（MCU，4s）
S4_F03 陆景琛搂住承诺（MCU，4s）
S4_F04 苏念核心台词（MCU，4s）
S4_F05 院子全景收尾（WIDE，4s）
```

5 个镜头，前 4 个空间没变、轴线没变、只是台词推进。
应合并为：
```
S4_F01（合并 F01-F03）三人晒太阳到女儿说出"我最喜欢的人"，陆景琛搂住承诺（6-8s）
S4_F02（合并 F04）苏念挽陆景琛手臂说："陆景琛，这辈子，我信你"（5-6s）
S4_F03（保留 F05）院子全景依偎收尾（4-5s）
```

3 个镜头替代 5 个，节奏更自然。

**反例：第 47 集场景 3 对峙**

```
S3_F01 陆鸿文攥着文件手抖（MCU）
S3_F02 陆景琛敲桌展示（MCU，2人）
S3_F03 文件特写（CU）
S3_F04 陆鸿文抬头（MCU，2人）
S3_F05 陆景琛对视（MCU，2人）
```

5 个镜头。S3_F02 / F04 / F05 是同一空间同一对峙的 3 段对话，
应合并为 1 个 6-8s 的"陆景琛敲桌→陆鸿文抬头→对视"长镜头，
只在 S3_F03（文件特写）保留一个 3-4s 揭示切。

最终 3 个镜头替代 5 个。

**跨镜头衔接**：上一镜头的结束动作必须在本镜头持续或在本镜头内重新建立，禁止凭空出现新道具/新动作。例如S1_F01必须完成"摸出照片"动作，S1_F02的"手攥照片"才有依据。

### 8. 叙事约束

- 一镜到底，禁止无镜头切换
- 禁止添加原文没有的台词

## 第二部分：镜头语言选择指南

### 景别选择

| 句子功能 | 推荐景别 | 典型场景 |
|---------|---------|---------|
| 建场/交代 | 远景或中景 | 场景开篇，展示空间 |
| 对话/动作 | 中景 | 两人对话、行走 |
| 情绪/反转 | 近景 | 表情变化、真相揭示 |
| 爆点/揭示 | 推进 + 近景 | 打脸、身份暴露 |
| 细节/心理 | 特写 | 眼神、手部、关键物品 |
| 收尾/抽离 | 拉远或空镜 | 人物远去、环境空镜 |

### 情绪→特写强度映射

情绪越强烈，特写越多、越近、切换越快：
- 平静对话 → 中景为主，少量近景，3-5秒切一次
- 紧张争执 → 近景+特写交替，2-3秒切一次
- 愤怒爆点 → 大特写快速切换，1-2秒切一次，面部细节占满画面

### 运镜选择

每镜头只选1种，禁止推拉摇移同时堆叠。

| 表达目的 | 运镜方式 | 典型场景 |
|---------|---------|---------|
| 清晰传达 | 固定镜头 | 信息揭示、关键对白 |
| 情绪升温 | 缓慢推进 | 逼近真相、压力增加 |
| 失落抽离 | 缓慢拉远 | 结尾收束、孤独感 |
| 空间移动 | 跟拍/平移 | 走路、穿越空间 |
| 视角切换 | POV/过肩 | 角色所见、对话交流 |
| 无特殊目的 | 固定（默认） | — |

### 拍摄角度

- 平视（默认）/ 俯拍（压抑）/ 仰拍（力量）
- 禁止在 prompt 中写焦段（35mm/50mm/85mm）和光圈（f1.4/f2.8）等相机参数

### 构图

- 单人：三分法，人物在左或右 1/3 处，视线方向留空
- 双人：对称或过肩，各占一侧
- 爆点：人物居中推进，背景简洁
- 必须标注角色在画面的横向和纵向位置

### 画面主体数量限制

每个镜头最多 2-3 个有台词/有表情的主体人物。
如有更多在场者，放在后景或画外，不占据画面主要空间。

### 旁观者自然行为

当一个人在说话时，在场的其他人不能呆板站立。
必须描写至少一种微动作：调整站姿、转头、整理衣角、看手机、抿嘴、
眼神飘向别处、微表情变化等。旁观者的存在感通过这些小动作体现。

### 路人/边缘角色规则

路人（保安、服务员、路人甲等）每个镜头中出现时
长相和造型必须各不相同。随机搭配不同的发型、体型、服装颜色，避免重复面孔。

## 第三部分：光影系统

### 三点布光（每个镜头指定）

- **主光（Key Light）**：主要照明方向和强度
- **辅助光（Fill Light）**：阴影补充，控制反差
- **轮廓光（Rim Light）**：后方勾勒主体轮廓

### 光质

| 类型 | 效果 | 适用场景 |
|------|------|---------|
| 硬光 | 正午阳光、聚光灯 → 戏剧性、高对比 | 冲突、揭示 |
| 柔光 | 阴天、柔光箱 → 柔和、自然 | 日常、温情 |
| 侧光 | 强化面部立体感 | 人物特写 |
| 逆光 | 剪影、轮廓光效果 | 悬念、氛围 |

### 色调

| 类型 | 效果 |
|------|------|
| 暖色调（金色/橙色） | 温馨、怀旧 |
| 冷色调（蓝色/青色） | 冷酷、紧张 |
| 低饱和 | 柔和、压抑、纪实 |
| 高饱和 | 强烈、鲜活、情绪化 |

## 第四部分：情绪→镜头映射（含面部引导）

| 情绪 | 景别 | 面部引导 | 运镜 | 光线 |
|------|------|---------|------|------|
| 孤独 | 远景留白 | 目光向下或望向窗外 | 静止或极缓慢 | 冷色调 |
| 压迫 | 中近景居中 | 呼吸短促 | 推进 | 强反差光、低角度 |
| 焦虑 | 特写碎剪 | 手指无意识活动、眼神快速游移 | — | 冷色 |
| 愤怒 | 近景正面 | 咬紧牙关、鼻孔翕动 | 快速推进 | 硬光侧光 |
| 悲伤 | 中景或近景 | 眼眶泛红、喉结滚动、嘴唇微颤 | 固定 | 柔光浅景深 |
| 决心 | 中近景正面 | 眼神锐利坚定、下巴微抬、呼吸均匀深长 | 固定 | 硬光 |
| 恐惧 | 近景 | 瞳孔放大、面色苍白、手指发凉 | 手持晃动 | 暗调逆光 |
| 轻蔑 | 近景 | 嘴角一侧上扬、目光从下往上扫、头微偏 | 固定 | — |
| 喜悦 | 近景 | 眼角细纹、苹果肌上提、呼吸变轻快 | 固定 | 暖光 |
| 平静 | 中景 | — | 固定 | 柔光、暖色调 |

## 第五部分：连续性规则

### 同场景连续镜头

- 人物位置/姿势从上一镜头结束状态自然延续
- 光线环境（时间、天气、光源方向、色温）必须一致
- 场景布置（家具、道具、背景）必须一致
- 角色着装必须完全相同

### 场景切换

- 禁止 A→C 直接跳跃，必须 A→过渡→B
- 过渡镜头必须体现空间移动（走廊/电梯/街道/楼梯）

### 角色状态延续

- 伤势、情绪、能量、位置必须从前序继承
- 变化必须有明确原因

---

## 第六部分：禁忌

- 禁止抽象词："温暖治愈、氛围感、宿命感、高级感"及英文 cinematic、film grain 等
- 禁止对白篡改：引号内对白原样保留
- 禁止自创角色或添加原文没有的台词
- 禁止散文风格：用执行口吻写镜头怎么拍、人物做什么
- 禁止全片同景别：信息句中景、情绪句近景、氛围句远景，必须变化
- 禁止无意义运镜：没有表达目的时用固定镜头
- 禁止空泛描述："美丽的风景" → "金色麦田在夕阳下泛起波浪"
- **禁止要求生成文字/数字**：Seedance 不擅长生成可读文字，用视觉元素替代
  - ✗ "屏幕显示'播放量10w+'"
  - ✓ "手机屏幕亮起，彩色光条闪烁"
- **禁止 camera 参数**：Seedance 不识别焦段、光圈（35mm/f1.4等）

---

## 第七部分：质量控制与约束词

每个镜头末尾必须追加以下约束词（Seedance 必须项）：

- 高清，细节丰富，电影质感
- 保持无字幕
- 不要生成水印
- 不要生成Logo
- 视频全程禁止出现外形、着装、配饰完全一致的人物，禁止生成同款分身、双胞胎效果

# 家庭伦理剧 — 冲突模板

## 核心冲突类型

### 代际冲突
- 传统 vs 现代价值观（婚姻观/职业选择/育儿方式）
- 父母偏心引发的兄弟姐妹矛盾
- 养老责任分配

### 婚姻冲突
- 婆媳矛盾（核心高频冲突）
- 出轨/信任危机
- 经济权力不平衡
- 子女教育分歧

### 家族利益冲突
- 财产分配/继承权争夺
- 家族企业控制权
- 亲情与利益的撕扯

## 冲突节奏

```
每集：1个即时冲突（可在本集或下集解决）
每5集：1个积累爆发（长期压抑后的集中释放）
每10集：1个关系转折（某些关系彻底改变）
```

## 情感爆点触发条件

1. 秘密被发现（藏了10集以上的秘密突然曝光）
2. 立场反转（一直站在一方的人突然倒戈）
3. 牺牲揭示（某人的默默付出被偶然发现）
4. 亲情考验（在利益和亲情之间的终极选择）

# short-drama-writer

AI短剧/短视频生成 Claude Code 插件。

## 安装

```
/plugin marketplace add VeiLei/short-drama-writer
/plugin install short-drama-writer
```

## 技能

| 命令 | 说明 |
|------|------|
| `/drama-init` | 项目初始化 |
| `/drama-plan` | 大纲规划 |
| `/drama-write` | 剧本写作 |
| `/drama-review` | 审查润色 |
| `/drama-generate` | 视觉素材生成 |
| `/drama-query` | 项目查询 |
| `/drama-dashboard` | 可视化面板 |

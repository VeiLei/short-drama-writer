# 平台标准

## 抖音短剧

| 属性 | 约束 |
|------|------|
| 单集时长 | 1-3分钟 |
| 画面比例 | 9:16 竖屏 |
| 分辨率 | 1080x1920 |
| 前3秒 | 必须有钩子（冲突/悬念/反差） |
| 结尾 | 强悬念钩子（引导下一集） |
| 字幕 | 必须大字幕（屏幕中下1/3） |

## 快手短剧

| 属性 | 约束 |
|------|------|
| 单集时长 | 1-5分钟 |
| 画面比例 | 9:16 竖屏 |
| 分辨率 | 1080x1920 |
| 风格 | 更接地气，强冲突优先 |

## 微短剧平台（小程序短剧）

| 属性 | 约束 |
|------|------|
| 单集时长 | 1-2分钟 |
| 画面比例 | 9:16 |
| 集数 | 100集+ |
| 节奏 | 极快，每30秒一个爽点 |
| 前10集 | 免费试看，需要极高转化率 |
